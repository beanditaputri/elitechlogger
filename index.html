DASHBOARD DATA LOGGER ELITECH

SELAMAT DATANG!

Dashboard ini dirancang untuk memvisualisasikan hasil studi pemantauan suhu (Temperature) dan kelembapan (Humidity) yang dicatat menggunakan perangkat Data Logger Elitech. Dashboard ini membantu tim Quality Control dalam menganalisis stabilitas kondisi lingkungan, mengidentifikasi fluktuasi ekstrem, dan memastikan standar penyimpanan objek studi tetap terjaga sesuai parameter yang ditentukan.

Panduan Penggunaan

  1. Navigasi Halaman: Gunakan menu di sisi kiri untuk berpindah antara Home dan Analisis Data (Analytics).
  2. Filter Data: Atur rentang tanggal, bulan, tahun, dan pilih nomor Data Logger Elitech melalui menu kontrol di Analytics.
  3. Detail Data: Arahkan kursor pada titik grafik di Analytics untuk melihat nilai spesifik pada waktu tertentu.
  4. Open Google Sheet: Klik tombol “Open Google Sheet” di Analytics untuk mengisi data melalui Google Sheets.
  5. Export Data: Klik tombol “Export Data” di Analytics untuk mengunduh ringkasan data dalam format Excel.
